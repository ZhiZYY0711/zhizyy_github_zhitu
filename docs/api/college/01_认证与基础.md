# API 接口设计 - 高校端 (College Client) - 认证与基础

> **文档说明**: 高校端-认证与基础服务 API
> **服务**: `auth-service`, `user-service`
> **Base URL**: `/api/{service}/v1`

## 1. 认证服务 (Auth)

> **注意**: 所有端统一使用 `POST /api/auth/v1/login`，通过 `role` 字段区分端类型。
> 完整接口定义见 `student/01_认证与基础.md` 第 1 节，此处仅说明高校端特有的 Response 差异。

### 1.1 统一登录接口（高校端 Response）
- **URL**: `POST /api/auth/v1/login`
- **Request Body**:
  ```json
  {
    "role": "college",
    "login_type": "account",
    "username": "teacher_01",
    "password": "encrypted_xxx",
    "captcha_code": "x7z9"
  }
  ```
  > `captcha_code` 图形验证码，防止爆破，高校端必填。
- **Response**:
  ```json
  {
    "access_token": "eyJ...",
    "refresh_token": "...",
    "expires_in": 7200,
    "user_info": {
      "id": "30001",
      "name": "李老师",
      "role": "college",
      "sub_role": "counselor",
      "college_id": "col_001",
      "department_id": "dept_002"
    }
  }
  ```
  > `sub_role` 枚举: `counselor`(辅导员) | `dean`(系主任) | `admin`(教务)

### 1.2 刷新 Token / 登出
> 同 `student/01_认证与基础.md` 1.2、1.3 节，接口完全一致。

### 1.3 获取当前权限菜单
- **URL**: `GET /api/auth/v1/menus/tree`
- **描述**: 根据教职工角色返回动态菜单树。
- **Response**:
  ```json
  [
    {
      "path": "/dashboard",
      "name": "Dashboard",
      "title": "管理驾驶舱",
      "icon": "dashboard",
      "children": []
    },
    {
      "path": "/students",
      "name": "StudentMgr",
      "title": "学生管理",
      "permission": "student:view"
    }
  ]
  ```

## 2. 基础服务 (User/Basic)

### 2.1 获取教职工个人档案
- **URL**: `GET /api/user/v1/college/profile`
- **Response**:
  ```json
  {
    "id": "30001",
    "job_number": "T2010001",
    "name": "李老师",
    "title": "副教授",
    "phone": "13900000000",
    "email": "li@edu.com",
    "managed_classes": ["class_01", "class_02"]
  }
  ```

### 2.2 修改密码
- **URL**: `PUT /api/user/v1/account/password`
- **Request Body**:
  ```json
  {
    "old_password": "...",
    "new_password": "..."
  }
  ```

### 2.3 获取全校学院/专业/班级级联数据（字典）
- **URL**: `GET /api/user/v1/basic/organization-tree`
- **描述**: 用于前端级联选择器（学院 → 专业 → 班级）。
- **Response**:
  ```json
  [
    {
      "id": "col_001",
      "label": "计算机学院",
      "children": [
        {
          "id": "major_001",
          "label": "软件工程",
          "children": [
            {"id": "class_01", "label": "软工2101班"}
          ]
        }
      ]
    }
  ]
  ```
