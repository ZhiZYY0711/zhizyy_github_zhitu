# API 接口设计 - 高校端 (College Client) - 认证与基础

> **文档说明**: 高校端-认证与基础服务 API
> **服务**: `auth-service`, `user-service`
> **Base URL**: `/api/{service}/v1`

## 1. 认证服务 (Auth)

### 1.1 教职工登录
- **URL**: `POST /api/auth/v1/login/college`
- **描述**: 高校教职工（辅导员、教务处、院系领导）登录接口。
- **Request Body**:
  ```json
  {
    "username": "teacher_01",
    "password": "encrypted_xxx",
    "captcha_code": "x7z9" // 图形验证码，防止爆破
  }
  ```
- **Response**:
  ```json
  {
    "access_token": "eyJ...",
    "user_info": {
      "id": "t_1001",
      "name": "李老师",
      "role": "counselor", // counselor(辅导员), dean(系主任), admin(教务)
      "college_id": "col_001", // 所属学院
      "department_id": "dept_002" // 所属系部
    }
  }
  ```

### 1.2 获取当前权限菜单
- **URL**: `GET /api/auth/v1/menus/tree`
- **描述**: 根据教职工角色返回动态菜单树。
- **Response**:
  ```json
  [
    {
      "path": "/dashboard",
      "name": "Dashboard",
      "title": "管理驾驶舱",
      "icon": "dashboard",
      "children": [...]
    },
    {
      "path": "/students",
      "name": "StudentMgr",
      "title": "学生管理",
      "permission": "student:view"
    }
  ]
  ```

## 2. 基础服务 (User/Basic)

### 2.1 获取教职工个人档案
- **URL**: `GET /api/user/v1/college/profile`
- **Response**:
  ```json
  {
    "id": "t_1001",
    "job_number": "T2010001",
    "name": "李老师",
    "title": "副教授",
    "phone": "13900000000",
    "email": "li@edu.com",
    "managed_classes": ["class_01", "class_02"] // 管理的班级ID列表
  }
  ```

### 2.2 修改密码
- **URL**: `PUT /api/user/v1/account/password`
- **Request Body**:
  ```json
  {
    "old_password": "...",
    "new_password": "..."
  }
  ```

### 2.3 获取全校学院/专业/班级级联数据 (字典)
- **URL**: `GET /api/user/v1/basic/organization-tree`
- **描述**: 用于前端级联选择器 (学院 -> 专业 -> 班级)。
- **Response**:
  ```json
  [
    {
      "id": "col_001",
      "label": "计算机学院",
      "children": [
        {
          "id": "major_001",
          "label": "软件工程",
          "children": [
            {"id": "class_01", "label": "软工2101班"}
          ]
        }
      ]
    }
  ]
  ```
