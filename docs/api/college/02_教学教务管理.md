# API 接口设计 - 高校端 (College Client) - 教学教务管理

> **文档说明**: 高校端-教学教务中心 API
> **服务**: `user-service`, `training-service`
> **模块**: 1.3 高校管理驾驶舱 - 教学教务中心

## 1. 学生与班级管理

### 1.1 获取学生列表
- **URL**: `GET /api/user/v1/college/students`
- **Query**: 
  - `page=1`, `size=10`
  - `college_id=...`, `major_id=...`, `class_id=...`
  - `keyword=...` (姓名/学号)
  - `status=active` (在读/休学/毕业)
- **Response**:
  ```json
  {
    "total": 200,
    "records": [
      {
        "id": "stu_001",
        "student_no": "2021001",
        "name": "张三",
        "gender": "male",
        "class_name": "软工2101",
        "gpa": 3.8,
        "phone": "138...",
        "status": "active"
      }
    ]
  }
  ```

### 1.2 学生详情 (360视图)
- **URL**: `GET /api/user/v1/college/students/{id}/full-view`
- **Response**: 包含基础信息、学业成绩、实训记录、实习记录、预警历史的聚合视图。

### 1.3 批量导入学生
- **URL**: `POST /api/user/v1/college/students/import`
- **Content-Type**: `multipart/form-data`
- **Form Data**: `file` (Excel文件)
- **Response**:
  ```json
  {
    "success_count": 98,
    "fail_count": 2,
    "fail_url": "http://.../error_report.xlsx" // 失败行下载链接
  }
  ```

## 2. 实训教学管理

### 2.1 实训课程排期列表
- **URL**: `GET /api/training/v1/college/plans`
- **Query**: `semester=2024-1`
- **Response**:
  ```json
  [
    {
      "id": "plan_001",
      "course_name": "Java企业级开发实训",
      "start_date": "2024-07-01",
      "end_date": "2024-07-20",
      "target_majors": ["软件工程", "计算机科学"],
      "status": "published" // draft, published, ongoing, closed
    }
  ]
  ```

### 2.2 发布/编辑实训计划
- **URL**: `POST /api/training/v1/college/plans`
- **Request Body**:
  ```json
  {
    "course_name": "前端工程化实训",
    "description": "...",
    "credits": 2,
    "time_range": ["2024-07-01", "2024-07-15"],
    "visible_classes": ["class_01", "class_02"]
  }
  ```

### 2.3 导师分配管理
- **URL**: `POST /api/training/v1/college/mentors/assign`
- **描述**: 为班级或实训项目分配校内指导老师。
- **Request Body**:
  ```json
  {
    "scope_type": "class", // class(行政班), project_team(项目组)
    "scope_id": "class_01",
    "teacher_id": "t_1001",
    "role": "academic_advisor" // 学术导师
  }
  ```
