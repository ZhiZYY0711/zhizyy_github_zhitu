# API 接口设计 - 高校端 (College Client) - 就业监测与实习管理

> **文档说明**: 高校端-就业监测与实习全流程管理 API
> **服务**: `portal-college-service`, `internship-service`
> **模块**: 1.3 高校管理驾驶舱 - 就业监测大屏, 实习管理

## 1. 就业监测大屏 (Dashboard)

### 1.1 获取核心就业指标
- **URL**: `GET /api/portal-college/v1/dashboard/stats`
- **Query**: `year=2024`, `college_id=...` (可选，校级管理员看全校，院级看本院)
- **Response**:
  ```json
  {
    "total_graduates": 1000,
    "employment_rate": 0.85, // 就业率
    "internship_rate": 0.92, // 实习率
    "flexible_employment_rate": 0.15, // 灵活就业率
    "avg_salary": 8500,
    "top_industries": [
      {"name": "互联网", "ratio": 0.4},
      {"name": "金融", "ratio": 0.2}
    ]
  }
  ```

### 1.2 就业/实习趋势图
- **URL**: `GET /api/portal-college/v1/dashboard/trends`
- **Query**: `dimension=month` (month/week)
- **Response**:
  ```json
  {
    "labels": ["1月", "2月", "3月", "4月", "5月"],
    "series": [
      {"name": "实习率", "data": [0.1, 0.3, 0.6, 0.8, 0.9]},
      {"name": "三方签约率", "data": [0.05, 0.1, 0.2, 0.4, 0.6]}
    ]
  }
  ```

## 2. 实习过程管理

### 2.1 实习生状态列表
- **URL**: `GET /api/internship/v1/college/students`
- **Query**: `status=interning` (seeking/interning/completed), `company_name=...`
- **Response**:
  ```json
  {
    "records": [
      {
        "student_name": "张三",
        "company": "字节跳动",
        "position": "后端开发",
        "start_date": "2024-03-01",
        "mentor_name": "李导师",
        "last_report_time": "2024-05-20",
        "status": "normal" // normal, warning
      }
    ]
  }
  ```

### 2.2 三方协议/实习协议审核
- **URL**: `GET /api/internship/v1/college/contracts/pending`
- **描述**: 获取待审核的协议列表。
- **Response**: List of contracts.

- **URL**: `POST /api/internship/v1/college/contracts/{id}/audit`
- **Request Body**:
  ```json
  {
    "result": "pass", // pass, reject
    "reject_reason": "公司盖章不清晰" // if reject
  }
  ```

### 2.3 云巡视管理
- **URL**: `POST /api/internship/v1/college/inspections`
- **描述**: 辅导员发起云巡视（视频通话/问卷回访）。
- **Request Body**:
  ```json
  {
    "student_ids": ["stu_001", "stu_002"],
    "type": "questionnaire", // video, questionnaire, phone
    "template_id": "tpl_monthly_check"
  }
  ```

### 2.4 查看学生周报
- **URL**: `GET /api/internship/v1/college/reports`
- **Query**: `student_id=...`, `week=...`
- **Response**: 周报详情及企业导师评语。

### 2.5 导出就业数据报表
- **URL**: `GET /api/portal-college/v1/reports/export`
- **Query**: `type=excel`
- **Response**: Blob (Excel file).
