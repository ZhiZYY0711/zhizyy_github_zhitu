# API 接口设计 - 学生端 (Student Client) - 智能实训项目

> **文档说明**: 学生端-实训项目相关 API
> **服务**: `training-service`
> **模块**: 2. 智能实训项目管理

## 1. 项目广场与申请

### 1.1 实训项目广场 (列表)
- **URL**: `GET /api/training/v1/projects`
- **Query**: `industry=finance` (行业), `tech_stack=java`, `difficulty=hard`, `keyword=...`
- **Response**:
  ```json
  {
    "records": [
      {
        "id": "proj_001",
        "name": "银行核心交易系统仿真",
        "provider": "某国有银行科技部", // 企业来源
        "difficulty": 4, // 1-5星
        "tech_stack": ["Java", "Oracle", "Redis"],
        "status": "recruiting", // recruiting(招募中), ongoing(进行中), finished
        "team_size": 5,
        "current_members": 2
      }
    ]
  }
  ```

### 1.2 项目详情
- **URL**: `GET /api/training/v1/projects/{id}`
- **Response**: 包含项目详细介绍、PRD文档链接、技术架构图、招募职位需求。

### 1.3 申请加入项目/职位
- **URL**: `POST /api/training/v1/projects/{id}/apply`
- **Request Body**:
  ```json
  {
    "role_id": "role_backend_01", // 申请的角色ID (e.g., 后端开发)
    "reason": "我有相关经验...",
    "resume_id": "resume_001" // 可选，关联简历
  }
  ```

## 2. 敏捷开发协同 (Scrum)

### 2.1 我的实训项目 - 敏捷看板
- **URL**: `GET /api/training/v1/my-project/sprints/current/tasks`
- **描述**: 获取当前Sprint的任务看板数据。
- **Response**:
  ```json
  {
    "sprint_name": "Sprint 3: 订单模块开发",
    "columns": {
      "todo": [
        {"id": "t_1", "title": "设计订单表结构", "assignee": "张三", "story_points": 3}
      ],
      "in_progress": [
        {"id": "t_2", "title": "开发创建订单接口", "assignee": "李四", "story_points": 5}
      ],
      "review": [],
      "done": []
    }
  }
  ```

### 2.2 拖拽更新任务状态
- **URL**: `PUT /api/training/v1/tasks/{taskId}/move`
- **Request Body**:
  ```json
  {
    "to_column": "in_progress", // todo, in_progress, review, done
    "sprint_id": "sp_03"
  }
  ```

### 2.3 提交代码关联 (Git)
- **URL**: `POST /api/training/v1/my-project/repo/link`
- **Request Body**:
  ```json
  {
    "repo_url": "https://gitea.zhitu.com/team1/project.git",
    "branch": "feature/order-service"
  }
  ```

### 2.4 发起/参与360度互评
- **URL**: `POST /api/training/v1/reviews/peer`
- **Request Body**:
  ```json
  {
    "project_id": "proj_001",
    "target_user_id": "10002", // 被评价队友ID
    "dimensions": {
      "contribution": 9, // 贡献度 1-10
      "cooperation": 8,  // 配合度
      "skill": 8         // 技术水平
    },
    "comment": "代码质量很高，但沟通需要加强"
  }
  ```
