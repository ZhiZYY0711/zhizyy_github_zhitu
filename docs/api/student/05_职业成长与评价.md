# API 接口设计 - 学生端 (Student Client) - 职业成长与评价

> **文档说明**: 学生端-评价、证书与徽章 API
> **服务**: `growth-service`
> **模块**: 4. 职业成长与评价

## 1. 评价与成绩

### 1.1 获取实训/实习成绩单
- **URL**: `GET /api/growth/v1/evaluations`
- **Response**:
  ```json
  {
    "training_score": 88.5,
    "internship_score": 92.0,
    "breakdown": {
      "enterprise_eval": 90,
      "school_eval": 85,
      "bonus": 5
    }
  }
  ```
