# API 接口设计 - 学生端 (Student Client) - 实习全流程

> **文档说明**: 学生端-实习求职与过程管理 API
> **服务**: `internship-service`
> **模块**: 3. 实习全流程管理

## 1. 求职与签约

### 1.1 实习岗位搜索
- **URL**: `GET /api/internship/v1/jobs`
- **Query**: `city=Beijing`, `salary_min=200`, `type=backend`
- **Response**: 职位列表，包含匹配度(match_score)。

### 1.2 投递简历
- **URL**: `POST /api/internship/v1/jobs/{jobId}/apply`
- **Request Body**:
  ```json
  {
    "resume_version": "v2_final"
  }
  ```

### 1.3 签署三方/实习协议
- **URL**: `POST /api/internship/v1/contracts/sign`
- **描述**: 这是一个复杂的业务流程，通常涉及调用第三方电子签服务。
- **Request Body**:
  ```json
  {
    "contract_id": "ctr_202405001",
    "signature_data": "base64_image_or_token", // 手写签名数据或CA认证Token
    "confirm_read": true
  }
  ```

## 2. 过程监控

### 2.1 LBS考勤打卡
- **URL**: `POST /api/internship/v1/attendance/clock-in`
- **Request Body**:
  ```json
  {
    "type": "in", // in(上班), out(下班)
    "location": {
      "lat": 39.9042,
      "lng": 116.4074,
      "address": "北京市朝阳区..."
    },
    "wifi_mac": "00:11:22:33:44:55", // 辅助防作弊
    "photo_url": "https://..." // 现场拍照(可选)
  }
  ```
- **Response**:
  ```json
  {
    "status": "normal", // normal, late, early_leave, location_error
    "server_time": "09:05:00"
  }
  ```

### 2.2 创建/保存周报
- **URL**: `POST /api/internship/v1/reports`
- **Request Body**:
  ```json
  {
    "week_number": 5,
    "start_date": "2024-05-20",
    "end_date": "2024-05-26",
    "content_done": "完成登录模块...",
    "content_plan": "下周进行支付对接...",
    "content_problem": "无",
    "mood_score": 4 // 1-5心情指数，用于情感分析
  }
  ```

### 2.3 获取我的周报列表 (含批阅状态)
- **URL**: `GET /api/internship/v1/reports/my`
- **Response**:
  ```json
  [
    {
      "id": "rpt_005",
      "week_number": 5,
      "status": "reviewed", // draft, submitted, reviewed
      "mentor_comment": "工作很扎实，继续保持",
      "teacher_comment": "注意文档规范"
    }
  ]
  ```
