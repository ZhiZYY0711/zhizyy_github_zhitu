# API 接口设计 - 学生端 (Student Client) - 认证与基础

> **文档说明**: 学生端-认证与基础服务 API
> **服务**: `auth-service`, `user-service`
> **Base URL**: `/api/{service}/v1`

## 1. 认证服务 (Auth)

### 1.1 学生登录 (多方式)
- **URL**: `POST /api/auth/v1/login/student`
- **描述**: 支持账号密码、手机验证码、微信小程序一键登录。
- **Request Body**:
  ```json
  {
    "login_type": "account", // 枚举: account, mobile, wechat_mini
    "username": "student_01", // account必填
    "password": "encrypted_xxx", // account必填
    "mobile": "13800000000", // mobile必填
    "sms_code": "123456", // mobile必填
    "wx_code": "091xxxx...", // wechat_mini必填 (js_code)
    "wx_encrypted_data": "...", // wechat_mini选填 (获取手机号用)
    "wx_iv": "..."
  }
  ```
- **Response**:
  ```json
  {
    "access_token": "eyJhbGciOi...",
    "refresh_token": "d7a8s7d8...",
    "expires_in": 7200,
    "user_info": {
      "id": "10001",
      "name": "张三",
      "avatar": "https://...",
      "role": "student",
      "college_id": "5001",
      "major_id": "6002"
    }
  }
  ```

## 2. 基础服务 (User)

### 2.1 获取个人基础信息
- **URL**: `GET /api/user/v1/student/profile`
- **描述**: 获取学生的基础学籍信息、联系方式等。
- **Response**:
  ```json
  {
    "id": "10001",
    "student_no": "2021001001", // 学号
    "name": "张三",
    "gender": "male",
    "college_name": "计算机学院",
    "major_name": "软件工程",
    "class_name": "2101班",
    "enrollment_year": 2021,
    "phone": "13800000000",
    "email": "zhangsan@edu.com",
    "skills_tags": ["Java", "React", "Spring Boot"] // 技能标签
  }
  ```

### 2.2 修改个人资料
- **URL**: `PUT /api/user/v1/student/profile`
- **Request Body**:
  ```json
  {
    "avatar": "https://new-avatar.url",
    "phone": "13900000000",
    "email": "new@edu.com",
    "skills_tags": ["Java", "Vue3", "Python"],
    "introduction": "热爱编程..."
  }
  ```
