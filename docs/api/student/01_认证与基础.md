# API 接口设计 - 学生端 (Student Client) - 认证与基础

> **文档说明**: 学生端-认证与基础服务 API
> **服务**: `auth-service`, `user-service`
> **Base URL**: `/api/{service}/v1`

## 1. 认证服务 (Auth)

> **注意**: 所有端统一使用 `POST /api/auth/v1/login`，通过 `role` 字段区分端类型。

### 1.1 统一登录接口
- **URL**: `POST /api/auth/v1/login`
- **描述**: 支持账号密码、手机验证码、微信小程序一键登录。`role` 字段决定登录身份类型。
- **Request Body**:
  ```json
  {
    "role": "student",
    "login_type": "account",
    "username": "student_01",
    "password": "encrypted_xxx",
    "mobile": "13800000000",
    "sms_code": "123456",
    "wx_code": "091xxxx...",
    "wx_encrypted_data": "...",
    "wx_iv": "..."
  }
  ```
  > `login_type` 枚举: `account`(账号密码) | `mobile`(手机验证码) | `wechat_mini`(微信小程序)
  > 开发阶段仅实现 `account` 类型，其余类型预留。

- **Response**:
  ```json
  {
    "access_token": "eyJhbGciOi...",
    "refresh_token": "d7a8s7d8...",
    "expires_in": 7200,
    "user_info": {
      "id": "10001",
      "name": "张三",
      "avatar": "https://...",
      "role": "student",
      "college_id": "5001",
      "major_id": "6002"
    }
  }
  ```

### 1.2 刷新 Token
- **URL**: `POST /api/auth/v1/token/refresh`
- **描述**: 使用 refresh_token 换取新的 access_token，无需重新登录。
- **Request Body**:
  ```json
  {
    "refresh_token": "d7a8s7d8..."
  }
  ```
- **Response**:
  ```json
  {
    "access_token": "eyJhbGciOi...",
    "expires_in": 7200
  }
  ```

### 1.3 登出
- **URL**: `POST /api/auth/v1/logout`
- **描述**: 服务端吊销 token（加入黑名单）。
- **Headers**: `Authorization: Bearer {access_token}`
- **Response**: `{ "message": "ok" }`

## 2. 基础服务 (User)

### 2.1 获取个人基础信息
- **URL**: `GET /api/user/v1/student/profile`
- **描述**: 获取学生的基础学籍信息、联系方式等。
- **Response**:
  ```json
  {
    "id": "10001",
    "student_no": "2021001001",
    "name": "张三",
    "gender": "male",
    "college_name": "计算机学院",
    "major_name": "软件工程",
    "class_name": "2101班",
    "enrollment_year": 2021,
    "phone": "13800000000",
    "email": "zhangsan@edu.com",
    "skills_tags": ["Java", "React", "Spring Boot"]
  }
  ```

### 2.2 修改个人资料
- **URL**: `PUT /api/user/v1/student/profile`
- **Request Body**:
  ```json
  {
    "avatar": "https://new-avatar.url",
    "phone": "13900000000",
    "email": "new@edu.com",
    "skills_tags": ["Java", "Vue3", "Python"],
    "introduction": "热爱编程..."
  }
  ```

### 2.3 修改密码
- **URL**: `PUT /api/user/v1/account/password`
- **Request Body**:
  ```json
  {
    "old_password": "...",
    "new_password": "..."
  }
  ```
