# API 接口设计 - 学生端 (Student Client) - 学生成长工作台

> **文档说明**: 学生端-工作台核心业务 API
> **服务**: `student-portal-service`
> **模块**: 1.2 学生成长工作台

## 1. 仪表盘 (Dashboard)

### 1.1 获取工作台聚合首页
- **URL**: `GET /api/student-portal/v1/dashboard`
- **描述**: 聚合核心指标、待办Top3、推荐Top3，用于首页快速渲染。
- **Response**:
  ```json
  {
    "stats": {
      "gpa": 3.8,
      "credit_completed": 120, // 已修学分
      "training_hours": 200, // 实训时长
      "internship_status": 1 // 0:未开始, 1:求职中, 2:实习中, 3:已结束
    },
    "radar_summary": { // 雷达图简略数据
      "labels": ["技术", "沟通", "管理", "文档", "创新", "学习"],
      "data": [80, 75, 60, 85, 70, 90]
    },
    "pending_tasks_count": 5
  }
  ```

### 1.2 获取能力雷达详情
- **URL**: `GET /api/student-portal/v1/capability/radar`
- **描述**: 获取详细的能力画像数据及同专业对比。
- **Response**:
  ```json
  {
    "dimensions": [
      {"key": "tech", "label": "技术能力", "score": 80, "max": 100},
      {"key": "comm", "label": "沟通协作", "score": 75, "max": 100},
      ...
    ],
    "peer_average": [70, 72, 65, 70, 68, 75], // 同专业平均分
    "history": [ // 历史趋势，用于折线图
      {"date": "2023-09", "avg_score": 70},
      {"date": "2024-03", "avg_score": 78}
    ]
  }
  ```

## 2. 任务与推荐

### 2.1 任务指挥中心 - 待办列表
- **URL**: `GET /api/student-portal/v1/tasks`
- **Query**: `status=pending` (pending/completed), `page=1`, `size=20`
- **Response**:
  ```json
  {
    "total": 15,
    "records": [
      {
        "id": "task_001",
        "type": "weekly_report", // 任务类型: weekly_report, interview, training_submit
        "title": "第5周实习周报待提交",
        "deadline": "2024-05-24T23:59:59",
        "priority": "high", // high, medium, low
        "jump_url": "/internship/reports/new?week=5" // 前端跳转路由
      },
      {
        "id": "task_002",
        "type": "interview_invite",
        "title": "字节跳动-后端开发实习生 面试邀约",
        "deadline": "2024-05-25T14:00:00",
        "priority": "high",
        "jump_url": "/internship/interviews/101"
      }
    ]
  }
  ```

### 2.2 个性化推荐流

- **Query**: `type=all` (all/project/job/course)
