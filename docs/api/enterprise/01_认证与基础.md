# API 接口设计 - 企业端 (Enterprise Client) - 认证与基础

> **文档说明**: 企业端-认证与基础服务 API
> **服务**: `auth-service`, `user-service`
> **Base URL**: `/api/{service}/v1`

## 1. 认证服务 (Auth)

### 1.1 企业员工登录
- **URL**: `POST /api/auth/v1/login/enterprise`
- **Request Body**:
  ```json
  {
    "username": "hr_01",
    "password": "encrypted_xxx"
  }
  ```
- **Response**:
  ```json
  {
    "access_token": "...",
    "user_info": {
      "id": "u_2001",
      "name": "王HR",
      "role": "hr", // hr, mentor(导师), admin(企业管理员)
      "enterprise_id": "ent_001"
    }
  }
  ```

## 2. 企业基础信息 (Basic)

### 2.1 获取企业详情
- **URL**: `GET /api/user/v1/enterprise/profile`
- **Response**:
  ```json
  {
    "id": "ent_001",
    "name": "字节跳动",
    "logo": "https://...",
    "description": "...",
    "industry": "互联网",
    "scale": "10000人以上",
    "address": "北京市海淀区...",
    "verification_status": "verified" // verified, pending, rejected
  }
  ```

### 2.2 更新企业信息
- **URL**: `PUT /api/user/v1/enterprise/profile`
- **Request Body**:
  ```json
  {
    "description": "新的简介...",
    "logo": "new_url"
  }
  ```

### 2.3 员工管理 (管理员)
- **URL**: `GET /api/user/v1/enterprise/staff`
- **Response**: 员工列表。

- **URL**: `POST /api/user/v1/enterprise/staff`
- **描述**: 添加新员工（如导师）。
- **Request Body**:
  ```json
  {
    "username": "mentor_01",
    "name": "李技术",
    "role": "mentor",
    "email": "li@byte.com"
  }
  ```
