# API 接口设计 - 企业端 (Enterprise Client) - 招聘与选拔

> **文档说明**: 企业端-招聘流程管理 API
> **服务**: `portal-enterprise-service`, `internship-service`
> **模块**: 1.4 企业人才工作台 - 招聘与选拔中心

## 1. 职位管理

### 1.1 发布新职位
- **URL**: `POST /api/internship/v1/enterprise/jobs`
- **Request Body**:
  ```json
  {
    "title": "Java实习生",
    "type": "internship",
    "description": "...",
    "requirements": "熟悉Spring Boot...",
    "salary_range": "200-300/天",
    "location": "北京"
  }
  ```

### 1.2 职位列表管理
- **URL**: `GET /api/internship/v1/enterprise/jobs`
- **Query**: `status=active` (active/closed)

## 2. 简历与面试

### 2.1 候选人简历列表
- **URL**: `GET /api/internship/v1/enterprise/applications`
- **Query**: `job_id=...`, `status=pending` (pending/interviewing/offer/rejected)
- **Response**:
  ```json
  {
    "records": [
      {
        "application_id": "app_001",
        "student_name": "张三",
        "school": "XX大学",
        "resume_url": "http://...",
        "match_score": 0.95, // 智能匹配度
        "apply_time": "2024-05-20"
      }
    ]
  }
  ```

### 2.2 发起面试邀约
- **URL**: `POST /api/internship/v1/enterprise/interviews`
- **Request Body**:
  ```json
  {
    "application_id": "app_001",
    "time": "2024-05-25 14:00:00",
    "type": "online", // online, offline
    "link": "https://meeting.tencent.com/..."
  }
  ```

### 2.3 人才蓄水池 (Talent Pool)
- **URL**: `POST /api/portal-enterprise/v1/talent-pool/collect`
- **描述**: 将优秀但不合适的候选人加入人才库。
- **Request Body**: `{"student_id": "stu_001", "tags": ["潜力股", "25届"]}`
