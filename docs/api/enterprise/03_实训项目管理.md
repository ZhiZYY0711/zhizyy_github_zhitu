# API 接口设计 - 企业端 (Enterprise Client) - 实训项目管理

> **文档说明**: 企业端-企业实训项目发布与管理 API
> **服务**: `training-service`
> **模块**: 2. 智能实训项目管理

## 1. 项目资源库

### 1.1 发布企业实训项目
- **URL**: `POST /api/training/v1/enterprise/projects`
- **Request Body**:
  ```json
  {
    "name": "电商高并发秒杀系统",
    "description": "基于真实业务场景...",
    "difficulty": 4,
    "tech_stack": ["Spring Cloud", "Redis", "RocketMQ"],
    "max_teams": 10,
    "resources_url": "http://oss..." // 脱敏数据/文档包
  }
  ```

### 1.2 项目团队监控
- **URL**: `GET /api/training/v1/enterprise/projects/{id}/teams`
- **Response**: 查看参与该项目的学生团队列表及其当前进度。

## 2. 资源配置

### 2.1 分配企业导师
- **URL**: `POST /api/training/v1/enterprise/projects/{id}/mentors`
- **Request Body**:
  ```json
  {
    "mentor_id": "u_2002", // 员工ID
    "team_ids": ["team_01", "team_02"] // 负责指导的团队
  }
  ```
