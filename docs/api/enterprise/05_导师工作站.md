# API 接口设计 - 企业端 (Enterprise Client) - 导师工作站

> **文档说明**: 企业端-企业导师专属功能 API
> **服务**: `portal-enterprise-service`, `training-service`
> **模块**: 1.4 企业人才工作台 - 导师工作站

## 1. 导师仪表盘

### 1.1 获取导师待办概览
- **URL**: `GET /api/portal-enterprise/v1/mentor/dashboard`
- **Response**:
  ```json
  {
    "pending_code_reviews": 5, // 待代码评审
    "pending_weekly_reports": 3, // 待周报批阅
    "upcoming_interviews": 1
  }
  ```

## 2. 指导与评审

### 2.1 代码评审 (Code Review)
- **URL**: `GET /api/training/v1/mentor/code-reviews`
- **Query**: `status=pending`
- **Response**: 待评审的 PR/Merge Request 列表。

- **URL**: `POST /api/training/v1/mentor/code-reviews/{id}/comment`
- **Request Body**:
  ```json
  {
    "file": "UserService.java",
    "line": 45,
    "comment": "建议使用Stream API优化此循环"
  }
  ```

### 2.2 批阅周报
- **URL**: `POST /api/internship/v1/mentor/reports/{id}/review`
- **Request Body**:
  ```json
  {
    "score": 90,
    "comment": "进度符合预期，下周注意并发问题"
  }
  ```

### 2.3 实习生最终评价
- **URL**: `POST /api/growth/v1/evaluations/enterprise`
- **Request Body**:
  ```json
  {
    "student_id": "stu_001",
    "scores": {
      "skill": 90,
      "teamwork": 85,
      "potential": 95
    },
    "hire_recommendation": "strongly_recommend" // 强烈推荐转正
  }
  ```
