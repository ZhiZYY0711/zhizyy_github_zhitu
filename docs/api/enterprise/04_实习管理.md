# API 接口设计 - 企业端 (Enterprise Client) - 实习管理

> **文档说明**: 企业端-实习生入职与日常管理 API
> **服务**: `internship-service`
> **模块**: 3. 实习全流程管理

## 1. 入职与合约

### 1.1 发送实习Offer/签约邀请
- **URL**: `POST /api/internship/v1/enterprise/offers`
- **Request Body**:
  ```json
  {
    "application_id": "app_001",
    "salary": "250/天",
    "start_date": "2024-06-01",
    "contract_template_id": "tpl_001"
  }
  ```

### 1.2 查看签约状态
- **URL**: `GET /api/internship/v1/enterprise/contracts`
- **Query**: `status=signed`

## 2. 日常管理

### 2.1 实习生列表
- **URL**: `GET /api/internship/v1/enterprise/interns`
- **Response**: 当前在岗实习生列表。

### 2.2 考勤审批
- **URL**: `POST /api/internship/v1/enterprise/attendance/audit`
- **Request Body**:
  ```json
  {
    "record_id": "att_001",
    "action": "approve", // approve, reject
    "comment": "同意"
  }
  ```

### 2.3 实习证明发放
- **URL**: `POST /api/internship/v1/enterprise/certificates/issue`
- **Request Body**:
  ```json
  {
    "intern_id": "stu_001",
    "comment": "表现优秀，欢迎转正"
  }
  ```
