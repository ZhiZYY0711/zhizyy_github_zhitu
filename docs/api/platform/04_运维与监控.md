# API 接口设计 - 平台端 (Platform Client) - 运维与监控

> **文档说明**: 平台端-系统监控与日志审计 API
> **服务**: `monitor-service`, `system-service`
> **模块**: 1.5 平台运营管理后台 - 系统运维与监控

## 1. 仪表盘监控

### 1.1 系统健康状态
- **URL**: `GET /api/monitor/v1/health`
- **Response**:
  ```json
  {
    "cpu_usage": 0.45,
    "memory_usage": 0.60,
    "active_services": 12,
    "error_rate": 0.001,
    "online_users": 5230
  }
  ```

### 1.2 在线用户趋势
- **URL**: `GET /api/monitor/v1/users/online-trend`
- **Response**: 过去24小时在线人数曲线。

## 2. 日志审计

### 2.1 操作日志查询
- **URL**: `GET /api/system/v1/logs/operation`
- **Query**: 
  - `user_id=...`
  - `module=contract` (模块)
  - `status=fail` (失败操作)
  - `start_time=...`, `end_time=...`
- **Response**:
  ```json
  {
    "records": [
      {
        "id": "log_001",
        "user_name": "王HR",
        "action": "delete_contract",
        "ip": "192.168.1.1",
        "time": "2024-05-24 10:00:00",
        "result": "success"
      }
    ]
  }
  ```

### 2.2 安全预警日志
- **URL**: `GET /api/system/v1/logs/security`
- **Query**: `level=high`
- **Response**: 异常登录、频繁失败尝试等安全事件。
