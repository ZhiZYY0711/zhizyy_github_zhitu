# API 接口设计 - 平台端 (Platform Client) - 认证与系统管理

> **文档说明**: 平台端-管理员登录与系统基础管理 API
> **服务**: `auth-service`, `system-service`
> **Base URL**: `/api/{service}/v1`

## 1. 管理员认证 (Auth)

### 1.1 平台管理员登录
- **URL**: `POST /api/auth/v1/login/admin`
- **Request Body**:
  ```json
  {
    "username": "admin",
    "password": "encrypted_xxx",
    "mfa_code": "123456" // 如开启MFA
  }
  ```
- **Response**:
  ```json
  {
    "access_token": "...",
    "permissions": ["system:all", "audit:view"]
  }
  ```

## 2. 基础主数据管理 (Master Data)

### 2.1 行业/技能标签管理
- **URL**: `POST /api/system/v1/tags`
- **描述**: 维护全平台通用的行业标签、技术栈标签。
- **Request Body**:
  ```json
  {
    "category": "tech_stack", // industry, tech_stack
    "name": "Spring Cloud",
    "parent_id": "tag_java"
  }
  ```

### 2.2 证书模板管理
- **URL**: `POST /api/system/v1/certificates/templates`
- **Request Body**:
  ```json
  {
    "name": "实训结业通用证书",
    "background_url": "http://...",
    "elements_layout": "{...}" // 姓名、日期等坐标配置
  }
  ```
