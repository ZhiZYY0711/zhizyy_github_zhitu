# API 接口设计 - 平台端 (Platform Client) - 多租户与机构审核

> **文档说明**: 平台端-高校与企业入驻审核 API
> **服务**: `user-service`, `system-service`
> **模块**: 1.5 平台运营管理后台 - 多租户与机构审核

## 1. 高校租户管理

### 1.1 开通高校租户
- **URL**: `POST /api/system/v1/tenants/colleges`
- **Request Body**:
  ```json
  {
    "name": "XX大学",
    "domain": "xxu.zhitu.com", // 专属子域名
    "admin_username": "admin_xxu", // 初始管理员
    "admin_email": "admin@xxu.edu.cn",
    "max_students": 50000,
    "expire_date": "2030-01-01"
  }
  ```

### 1.2 高校列表查询
- **URL**: `GET /api/system/v1/tenants/colleges`
- **Query**: `status=active`

## 2. 企业入驻审核

### 2.1 待审核企业列表
- **URL**: `GET /api/system/v1/audits/enterprises`
- **Query**: `status=pending`
- **Response**:
  ```json
  {
    "records": [
      {
        "id": "ent_001",
        "name": "某科技公司",
        "license_url": "http://...", // 营业执照
        "apply_time": "2024-05-24"
      }
    ]
  }
  ```

### 2.2 审核操作
- **URL**: `POST /api/system/v1/audits/enterprises/{id}`
- **Request Body**:
  ```json
  {
    "action": "pass", // pass, reject
    "reject_reason": "营业执照模糊不清"
  }
  ```
