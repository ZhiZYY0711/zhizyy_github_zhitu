# API 接口设计 - 平台端 (Platform Client) - 全局资源调度

> **文档说明**: 平台端-实训资源与内容审核 API
> **服务**: `training-service`, `portal-platform-service`
> **模块**: 1.5 平台运营管理后台 - 全局资源调度

## 1. 实训项目审核

### 1.1 待审核项目列表
- **URL**: `GET /api/portal-platform/v1/audits/projects`
- **Query**: `provider_type=enterprise` (只看企业发布的)
- **Response**: 待审核的实训项目列表。

### 1.2 审核实训项目
- **URL**: `POST /api/portal-platform/v1/audits/projects/{id}`
- **描述**: 确保项目内容合规、无敏感信息。
- **Request Body**:
  ```json
  {
    "action": "pass",
    "quality_rating": "S", // S/A/B/C，优质项目可置顶
    "comment": "项目质量很高，推荐首页"
  }
  ```

## 2. 推荐位管理

### 2.1 设置首页推荐
- **URL**: `POST /api/portal-platform/v1/recommendations/banner`
- **Request Body**:
  ```json
  {
    "target_type": "project", // project, enterprise, course
    "target_id": "proj_001",
    "image_url": "http://...",
    "order": 1
  }
  ```

### 2.2 优质导师/课程置顶
- **URL**: `POST /api/portal-platform/v1/recommendations/top-list`
- **Request Body**:
  ```json
  {
    "list_type": "mentor",
    "item_ids": ["u_2001", "u_2002"]
  }
  ```
