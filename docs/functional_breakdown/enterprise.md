# 企业端功能拆解

## 1. 核心定位
面向HR及企业导师，侧重人才选拔、实训项目管理与实习指导。帮助企业降本增效，提前锁定优质人才。

## 2. 功能模块详情

### 2.1 企业人才工作台
**招聘与管理的统一工作台**
- **招聘与选拔中心**: 
  - 岗位发布与管理。
  - 简历智能筛选（基于画像匹配）。
  - 面试流程管理（支持在线面试/笔试邀约）。
- **实训/实习管理**: 
  - 实习生入职/离职办理。
  - 考勤审批、周报批阅。
  - 任务分配与绩效记录。
- **人才蓄水池**: 收藏优质潜力学生，建立企业专属的储备人才库。
- **导师工作站**: 企业导师专属界面，用于查看指导学生进度、答疑解惑及评分。

### 2.2 实训项目运营
**通过实训选拔人才**
- **项目发布**: 将企业真实案例脱敏后发布到资源池，设置技术栈与难度标签。
- **技术指导**:
  - **企业导师职责**:把控技术方向，进行Code Review（行级批注）。
  - **在线答疑**: 解答实训小组的技术难题。
- **成果评估**: 参与项目结项评审，对学生技能应用、团队协作进行打分。

### 2.3 实习生管理
**实习全周期的精细化管理**
- **双选互动**: 发起“意向邀约”，处理学生投递，在线签署电子协议。
- **过程辅导**:
  - **周报批阅**: 查看实习生周报，了解工作进展与心态。
  - **考勤管理**: 审批LBS打卡异常与请假申请。
- **沟通反馈**: 与校方老师、实习生保持群组沟通。

### 2.4 企业效益分析
**人才合作价值评估**
- **企业效益视图**:
  - **人才转化率**: 查看实习转正率统计与招聘成本节省测算。
  - **合作贡献度**: 评估实习生为企业创造的直接/间接价值。
  - **证书验证**: 扫码验证学生持有的区块链实训/实习证书真伪。
