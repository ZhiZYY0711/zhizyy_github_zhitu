# 高校端功能拆解

## 1. 核心定位
面向辅导员及教务管理人员，侧重教学管理、就业监测、数据分析与预警。帮助高校提升就业率与人才培养质量。

## 2. 功能模块详情

### 2.1 高校管理驾驶舱
**教务与就业管理的统一门户**
- **教学教务中心**: 
  - 班级/学生信息管理。
  - 实训课程排期与设置。
  - 导师分配管理（为班级/项目分配校内导师）。
- **就业监测大屏**: 实时透视各院系实习率、就业率、专业对口率及薪资分布。
- **企业CRM系统**: 
  - 合作企业全生命周期管理。
  - 企业资质审核与合作等级评定。
- **预警指挥室**: 
  - **自动预警**: 针对未就业、实习异常（缺勤、周报缺失）、位置异常的学生进行自动分级预警。
  - **干预记录**: 记录对预警学生的干预措施与结果。

### 2.2 实训教学管理
**监控实训过程与质量**
- **项目进度监控**: 查看各实训小组的Scrum看板进度与燃尽图。
- **指导与评审**:
  - **校内导师职责**: 负责学术规范指导与文档质量把控。
  - **在线答疑**: 对学生提交的文档/代码进行批注与答疑。
- **成绩评定**: 结合系统预评分与过程表现，进行校方评价打分。

### 2.3 实习过程监管
**保障实习安全与质量**
- **三方协同**:
  - **云巡视**: 发起视频巡视或问卷回访，系统自动记录留痕。
  - **沟通群组**: 自动加入学生-企业导师的沟通群，随时掌握实习动态。
- **异常处理**: 接收异常行为预警（如连续未打卡），及时联系学生处理。

### 2.4 数据决策支持 (BI)
**基于数据的教学改进**
- **校级决策视图**:
  - **人才培养质量报告**: 查看基于企业反馈反推的课程设置分析。
  - **就业趋势预测**: 查看基于历年数据的本届就业压力点预测。
