## 阶段一：基础软件安装

### 1.1 安装 Docker Desktop
- [ ] 访问 https://www.docker.com/products/docker-desktop/ 下载安装包
- [ ] 安装并启动 Docker Desktop
- [ ] 验证安装：`docker --version`（要求 20.10+）
- [ ] 验证 Docker Compose：`docker-compose --version`

### 1.2 安装 JDK 17
- [ ] 下载 JDK 17（推荐 Oracle JDK 或 OpenJDK）
  - Oracle: https://www.oracle.com/java/technologies/downloads/#java17
  - OpenJDK: https://jdk.java.net/17/
- [ ] 配置环境变量 `JAVA_HOME`
- [ ] 验证安装：`java -version`

### 1.3 安装 Maven
- [ ] 下载 Maven（推荐 3.8+）
- [ ] 配置环境变量 `MAVEN_HOME`
- [ ] 配置阿里云镜像源（settings.xml）加速依赖下载
- [ ] 验证安装：`mvn -v`

### 1.4 安装 IntelliJ IDEA
- [ ] 下载并安装 IntelliJ IDEA 2023.1+（推荐 Ultimate 版）
- [ ] 安装插件：
  - [ ] **Lombok**（必需）
  - [ ] **Alibaba Cloud Toolkit**（推荐）
  - [ ] **Spring Boot**（推荐）

### 1.5 安装 Git
- [ ] 下载并安装 Git 2.30+
- [ ] 配置用户名和邮箱
- [ ] 验证安装：`git --version`

## 阶段二：中间件服务部署（Docker Compose）

### 2.1 准备目录结构
```
f:\projects\zhitu\docker\
├── docker-compose.yml
├── postgres/
│   ├── data/          # 数据持久化目录
│   └── init/          # 初始化脚本目录
├── redis/
│   └── data/          # 数据持久化目录
├── nacos/
│   ├── logs/          # 日志目录
│   └── data/          # 数据持久化目录
└── minio/
    ├── data/          # 数据持久化目录
    └── config/        # 配置目录
```

### 2.2 创建数据目录
在项目根目录下执行：
```cmd
cd f:\projects\zhitu\docker
mkdir postgres\data postgres\init redis\data nacos\logs nacos\data minio\data minio\config
```

### 2.3 启动中间件服务
```cmd
cd f:\projects\zhitu\docker
docker-compose up -d
```

### 2.4 验证服务状态
```cmd
docker-compose ps
```

### 2.5 服务访问验证

| 服务 | 验证方式 | 预期结果 |
|------|----------|----------|
| **Nacos** | 访问 http://localhost:8848/nacos | 能打开 Nacos 控制台 |
| **PostgreSQL** | 使用客户端连接 localhost:5432 | 能连接，数据库名 `zhitu_cloud` |
| **Redis** | 使用 redis-cli 或客户端连接 | 能连接，密码 `zhitu_redis_pwd` |
| **MinIO** | 访问 http://localhost:9001 | 能打开 MinIO 控制台，账号 `minioadmin/minioadmin` |

## 阶段三：后端微服务项目初始化

### 3.1 创建项目目录结构
```
f:\projects\zhitu\backend\
├── zhitu-gateway/              # Spring Cloud Gateway 网关服务
├── zhitu-auth/                 # 认证授权服务 (Spring Authorization Server)
├── zhitu-common/               # 公共模块
│   ├── zhitu-common-core/
│   ├── zhitu-common-security/
│   └── zhitu-common-redis/
├── zhitu-modules/              # 业务微服务
│   ├── zhitu-system/           # 系统管理服务
│   ├── zhitu-college/          # 高校端服务
│   ├── zhitu-enterprise/       # 企业端服务
│   ├── zhitu-student/          # 学生端服务
│   └── zhitu-platform/         # 平台运营端服务
└── pom.xml                     # 父工程 POM
```

### 3.2 父工程 POM 配置要点

**Spring Boot 版本**: 3.2.x
**Spring Cloud 版本**: 2023.0.x
**Spring Cloud Alibaba 版本**: 2022.0.0.0

核心依赖：
- `spring-boot-starter-parent`
- `spring-cloud-dependencies`
- `spring-cloud-alibaba-dependencies`

### 3.3 网关服务 (zhitu-gateway) 配置

核心依赖：
- `spring-cloud-starter-gateway`
- `spring-cloud-starter-loadbalancer`
- `spring-cloud-starter-alibaba-nacos-discovery`
- `spring-cloud-starter-alibaba-nacos-config`
- `spring-boot-starter-data-redis-reactive`

### 3.4 认证服务 (zhitu-auth) 配置

核心依赖：
- `spring-boot-starter-oauth2-authorization-server`
- `spring-boot-starter-oauth2-resource-server`
- `spring-boot-starter-security`
- `jjwt-api`, `jjwt-impl`, `jjwt-jackson`
- `spring-cloud-starter-alibaba-nacos-discovery`

### 3.5 业务服务通用配置

每个业务服务都需要：
- `spring-boot-starter-web`
- `spring-cloud-starter-alibaba-nacos-discovery`
- `spring-cloud-starter-alibaba-nacos-config`
- `spring-cloud-starter-openfeign`
- `spring-cloud-starter-loadbalancer`
- `spring-cloud-starter-circuitbreaker-resilience4j` (熔断)
- `spring-boot-starter-data-jpa` 或 `mybatis-spring-boot-starter`
- `postgresql` 驱动
- `spring-boot-starter-data-redis`
- `spring-boot-starter-validation`

## 阶段四：Nacos 配置中心初始化

### 4.1 创建命名空间
在 Nacos 控制台创建以下命名空间：
- `dev` - 开发环境
- `test` - 测试环境
- `prod` - 生产环境

### 4.2 创建共享配置
在 `dev` 命名空间下创建：

**1. 数据库配置 `zhitu-db.yaml`**
```yaml
spring:
  datasource:
    type: com.zaxxer.hikari.HikariDataSource
    driver-class-name: org.postgresql.Driver
    url: jdbc:postgresql://localhost:15432/zhitu_cloud
    username: postgres
    password: 123456
    hikari:
      minimum-idle: 5
      maximum-pool-size: 20
      connection-timeout: 30000
```

**2. Redis 配置 `zhitu-redis.yaml`**
```yaml
spring:
  redis:
    host: localhost
    port: 6379
    password: 123456
    database: 0
    timeout: 10s
    lettuce:
      pool:
        min-idle: 0
        max-idle: 8
        max-active: 8
        max-wait: -1ms
```

**3. MinIO 配置 `zhitu-minio.yaml`**
```yaml
minio:
  endpoint: http://localhost:9000
  access-key: minioadmin
  secret-key: minioadmin
  bucket-name: zhitu-files
```

## 附录：常用命令速查

### Docker 命令
```bash
# 启动所有服务
docker-compose up -d

# 停止所有服务
docker-compose down

# 查看日志
docker-compose logs -f [service_name]

# 重启单个服务
docker-compose restart [service_name]
```

### Maven 命令
```bash
# 编译项目
mvn clean compile

# 打包项目
mvn clean package

# 跳过测试打包
mvn clean package -DskipTests

# 安装到本地仓库
mvn clean install
```

## 注意事项

1. **内存要求**：微服务架构较为消耗内存，建议开发环境至少 16GB 内存
2. **端口冲突**：确保 8080-8086、5432、6379、8848、9000-9001 端口未被占用
3. **防火墙**：确保 Docker 容器能够正常访问本地端口
4. **时区**：所有服务已配置为 Asia/Shanghai 时区
