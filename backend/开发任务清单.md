- [ ] 在 `dev` 命名空间下创建共享配置 `zhitu-minio.yaml`

### 1.3 数据库初始化
- [ ] 优化 `docker/postgres/init/01_init_schema.sql`（补充 growth_service schema 和新增表）
- [ ] 执行 SQL 初始化脚本（Docker 启动时自动执行，或手动连接执行）
- [ ] 验证所有 Schema 和表已创建：`\dn` 查看 schema，`\dt schema.*` 查看表
- [ ] 插入基础测试数据（平台管理员账号、测试高校租户、测试企业租户）

### 1.4 JWT 证书生成
- [ ] 在 `zhitu-auth/src/main/resources/certs/` 目录下生成 RSA 密钥对
  ```bash
  openssl genrsa -out jwt-private.pem 2048
  openssl rsa -in jwt-private.pem -pubout -out jwt-public.pem
  ```
- [ ] 验证证书文件存在且格式正确

### 1.5 Maven 依赖验证

- [ ] 启动 `zhitu-auth`，访问 http://localhost:8081/actuator/health 返回 UP

- [ ] 创建 MyBatis-Plus 分页配置 `MybatisPlusConfig`

#### 2.1.2 zhitu-common-security

### 2.2 认证服务开发（zhitu-auth）

#### 2.2.1 数据层
- [ ] 创建 `SysUser` 实体类，对应 `auth_center.sys_user` 表
- [ ] 创建 `SysTenant` 实体类，对应 `auth_center.sys_tenant` 表
- [ ] 创建 `SysRefreshToken` 实体类，对应 `auth_center.sys_refresh_token` 表
- [ ] 创建对应的 MyBatis-Plus Mapper 接口
- [ ] 创建 `SysUserRepository`（Service 层，封装用户查询逻辑）

#### 2.2.2 登录接口
- [ ] 创建 `AuthController`，实现以下接口：
  - [ ] `POST /api/auth/v1/login` — 统一登录
    - [ ] 接收 `LoginRequest`（role、login_type、username、password）
    - [ ] 根据 `role` 路由到对应的用户查询逻辑
    - [ ] 验证密码（BCrypt）
    - [ ] 生成 access_token 和 refresh_token
    - [ ] 将 refresh_token 哈希存入数据库 `sys_refresh_token`
    - [ ] 将 access_token 存入 Redis（key: `token:access:{userId}`，用于主动吊销）
    - [ ] 返回 `LoginResponse`（access_token、refresh_token、expires_in、user_info）
  - [ ] `POST /api/auth/v1/token/refresh` — 刷新 token
    - [ ] 验证 refresh_token 是否存在且未过期
    - [ ] 生成新的 access_token
    - [ ] 返回新 token
  - [ ] `POST /api/auth/v1/logout` — 登出
    - [ ] 从 Redis 删除 access_token
    - [ ] 从数据库删除 refresh_token 记录
- [ ] 创建 `LoginRequest` DTO（含参数校验注解）
- [ ] 创建 `LoginResponse` DTO
- [ ] 创建 `AuthService` 实现登录业务逻辑
- [ ] 配置 Spring Security，放行 `/api/auth/v1/login` 和 `/api/auth/v1/token/refresh`

#### 2.2.3 用户管理接口
- [ ] `GET /api/user/v1/account/info` — 获取当前用户基础信息（从 token 解析）
- [ ] `PUT /api/user/v1/account/password` — 修改密码

#### 2.2.4 安全配置
- [ ] 配置 `SecurityFilterChain`，设置白名单路由
- [ ] 配置密码编码器 `BCryptPasswordEncoder`
- [ ] 配置 CORS 跨域

```

第2周：2.4 系统服务 + 数据库初始化完善
第3周：2.5 学生服务（认证与工作台部分）
第4周：2.6 企业服务（招聘与实习管理）
第5周：2.7 高校服务
第6周：2.8 平台服务 + 2.9 成长评价服务
第7周：2.10 规范收尾 + 联调测试
```
