    - **后端**：推荐使用 Java (Spring Boot) 或 Node.js (NestJS)。
    - **数据库**：MySQL (关系型数据) + Redis (缓存)。
    - **其他**：Docker 部署、Git 版本控制等。

3.  **编写数据库设计文档 (`docs/2_数据库设计.md`)**

        - `project_base` (实训项目库)
        - `internship_process` (实习全流程记录)
        - `evaluation_record` (多维评价记录)

4.  **编写 API 接口设计文档 (`docs/3_API接口设计.md`)**

    - **实现形式**：在文档中包含简单的 HTML/Vue 代码片段或 Mermaid 流程图来展示界面逻辑和布局。
