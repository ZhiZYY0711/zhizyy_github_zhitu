# 🚀 快速开始 - Playwright 自动化测试

## 一键运行测试

### 1. 确保开发服务器运行中

```bash
cd frontend
npm run dev
```

### 2. 运行测试（推荐 UI 模式）

```bash
npm run test:e2e:ui
```

这会打开 Playwright 的可视化界面，你可以：
- ✅ 选择要运行的测试
- ✅ 实时查看测试执行
- ✅ 查看每一步的截图
- ✅ 调试失败的测试

### 3. 或者运行所有测试

```bash
npm run test:e2e
```

## 📊 查看测试结果

测试完成后会自动生成 HTML 报告：

```bash
npx playwright show-report
```

## 📸 自动截图

所有关键页面会自动截图，保存在：
```
frontend/e2e/screenshots/
```

## 🐛 调试测试

如果测试失败，使用调试模式：

```bash
npm run test:e2e:debug
```

这会打开 Playwright Inspector，可以逐步执行测试。

## 💡 提示

- 测试会自动启动开发服务器（如果未运行）
- 失败的测试会自动截图
- 可以在 `playwright.config.ts` 中调整配置
- 详细文档见 `e2e/README.md`
